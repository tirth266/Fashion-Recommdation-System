## ğŸ’¡ Customization Tips

### Change Primary Color
Replace `#a855f7` throughout `src/index.css`:
```css
/* Instead of purple, use your brand color */
background: linear-gradient(135deg, YOUR_COLOR, ...);
```

### Adjust Animation Speed
Edit animation duration in `src/index.css`:
```css
.animate-float {
  animation: float 3s ease-in-out infinite; /* Change 3s */
}
```

### Modify Glassmorphism Blur
Change blur amount in `.glass` class:
```css
.glass {
  backdrop-filter: blur(10px); /* Increase/decrease */
}
```

### Add More Animations
Create new `@keyframes` in `src/index.css` and use with classes.
